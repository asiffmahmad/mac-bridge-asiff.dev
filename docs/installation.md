# Installation & Deployment Guide

## Automated Backend Setup

The easiest way to install the Mac Bridge agent backend daemon on macOS is to run the automated installer.

1. **Verify Prerequisites**:
   Make sure you have JDK 21+ and Maven installed:
   ```bash
   java -version
   mvn -version
   ```

2. **Execute Installer**:
   From the repository root directory, run:
   ```bash
   chmod +x scripts/install.sh
   ./scripts/install.sh
   ```

3. **Retrieve Autogenerated Password**:
   On the first launch, the bridge generates a random admin password:
   ```bash
   cat ~/.mac-bridge/users.json
   ```

4. **Verify Service Logs**:
   The installer installs a launchd plist in `~/Library/LaunchAgents/` to keep the bridge running:
   ```bash
   tail -n 50 ~/.mac-bridge/bridge.log
   ```

## Secure Cloudflare Tunneling (Production Setup)

To allow the frontend (hosted on Vercel/Netlify) to communicate with the local agent securely without opening firewall ports:

1. **Install Cloudflared**:
   ```bash
   brew install cloudflared
   ```

2. **Authenticate and Route Tunnel**:
   ```bash
   cloudflared tunnel login
   cloudflared tunnel create mac-bridge
   ```

3. **Configure Routing**:
   Route a subdomain to your local bridge (running on port 8080):
   ```yaml
   # ~/.cloudflare/config.yml
   url: http://localhost:8080
   tunnel: <tunnel-uuid>
   credentials-file: /Users/<user>/.cloudflare/<tunnel-uuid>.json
   ```

4. **Run the Tunnel**:
   ```bash
   cloudflared tunnel run mac-bridge
   ```
   Provide the resulting public Cloudflare URL (e.g. `https://mac.mydomain.com`) in the frontend Login URL input field.

## Frontend Web Deployment

To compile the React-Vite web frontend console:

1. **Install Dependencies**:
   ```bash
   cd mac-bridge-web
   npm install
   ```

2. **Build static site**:
   ```bash
   npm run build
   ```

3. **Host**:
   Deploy the resulting `dist/` directory to Vercel, Netlify, or AWS S3.
   Configure the frontend settings inside the app console.
